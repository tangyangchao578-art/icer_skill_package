# 流片检查清单 (Tapeout Checklist)

流片前必须完成的所有检查项。

## 时序签收

### 建立时间

- [ ] 所有工艺角 WNS ≥ 0
- [ ] 典型工艺角有足够裕量 (> 100ps)
- [ ] 关键路径已优化
- [ ] 路径报告已审查

### 保持时间

- [ ] 所有工艺角保持时间满足
- [ ] 无保持时间违规
- [ ] 修复后重新验证

### 时钟

- [ ] 时钟树平衡
- [ ] 时钟 skew 满足要求 (< 100ps)
- [ ] 时钟 latency 满足要求
- [ ] 门控时钟正确

### OCV/POCV

- [ ] OCV 分析完成
- [ ] OCV 裕量足够
- [ ] POCV 分析完成（如果要求）

## 物理验证

### DRC

- [ ] DRC 零违反 或 waiver 已批准
- [ ] 所有 waiver 已文档化
- [ ] Foundry 已确认 waiver
- [ ] DRC 报告已保存

### LVS

- [ ] LVS 零违反
- [ ] 所有网络匹配
- [ ] 所有器件匹配
- [ ] 无短路、无开路
- [ ] LVS 报告已保存

### ERC

- [ ] ERC 零违反
- [ ] 无浮动节点
- [ ] 无未连接端口
- [ ] 电源地连接正确

### 天线检查

- [ ] 天线违反零
- [ ] 所有天线修复完成
- [ ] 天线报告已保存

### 密度检查

- [ ] 金属密度满足要求
- [ ] 填充正确
- [ ] 密度报告已保存

## 功耗分析

### 功耗

- [ ] 总功耗满足预算
- [ ] 峰值功耗满足要求
- [ ] 功耗报告已保存

### IR 降

- [ ] IR 降满足要求 (< 5% Vdd)
- [ ] 无 IR 热点
- [ ] IR 降报告已保存

### 电迁移

- [ ] 电迁移满足要求
- [ ] 无 EM 违规
- [ ] EM 报告已保存

## 面积和利用率

### 面积

- [ ] 总面积满足预算
- [ ] 核心利用率满足要求
- [ ] 面积报告已保存

### 布线拥塞

- [ ] 无布线拥塞热点
- [ ] 所有网络已布线
- [ ] 拥塞报告已保存

## 输出文件

### GDS/GDSII

- [ ] GDS 文件生成正确
- [ ] 层次正确
- [ ] GDS 可读取
- [ ] 备份保存

### 网表

- [ ] 最终网表生成
- [ ] 网表与 LVS 一致
- [ ] 网表格式正确

### 约束文件

- [ ] SDC 文件生成
- [ ] SPEF 文件生成
- [ ] SDF 文件生成
