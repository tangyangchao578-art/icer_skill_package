# RTL 代码审查清单

RTL 编码完成后必须进行的代码审查检查。

## 设计规范

### 文件组织

- [ ] 文件名与模块名一致
- [ ] 文件头注释完整（作者、日期、版本、功能描述）
- [ ] 模块职责单一、清晰
- [ ] 参数化设计，无硬编码常量

### 命名规范

- [ ] 信号名使用小写加下划线
- [ ] 参数名使用大写加下划线
- [ ] 时钟信号以 `clk` 开头
- [ ] 复位信号以 `rst` 开头，低有效加 `_n` 后缀
- [ ] 使能信号以 `en` 结尾
- [ ] 寄存器输出以 `r_` 或 `_reg` 前缀/后缀

## 时序设计

### 时钟域

- [ ] 明确标识所有时钟域
- [ ] 跨时钟域信号使用同步器
- [ ] 异步复位同步释放
- [ ] 门控时钟使用集成门控单元

### 复位设计

- [ ] 复位信号同步处理
- [ ] 复位状态定义明确
- [ ] 复位后寄存器初始值正确
- [ ] 复位有效性（高有效/低有效）统一

### 时序逻辑

- [ ] 使用非阻塞赋值 (<=)
- [ ] 单 always 块单时钟边沿
- [ ] 无锁存器推断
- [ ] 无组合逻辑反馈

## 组合逻辑

### 编码风格

- [ ] 使用阻塞赋值 (=)
- [ ] 完整的 case 语句（加 default）
- [ ] 完整的 if-else 结构
- [ ] 无不完整敏感列表

### 避免问题

- [ ] 无锁存器推断
- [ ] 无组合环路
- [ ] 无未驱动信号
- [ ] 无多驱动信号

## 低功耗设计

### 门控时钟

- [ ] 使能信号条件明确
- [ ] 使用集成门控单元
- [ ] 门控时钟插入率 > 90%

### 电源管理

- [ ] 空闲模块断电设计
- [ ] 电源域隔离正确
- [ ] 电平转换器正确插入

## 可综合性

### 语法检查

- [ ] 无不可综合结构（delay、initial 等）
- [ ] 无系统任务（$display、$random 等，仅仿真用）
- [ ] 循环边界为常数
- [ ] 函数无递归调用

### 综合约束

- [ ] 时钟约束完整
- [ ] 输入输出延迟合理
- [ ] 假路径标识正确
- [ ] 多周期路径标识正确

## 可测试性

### DFT 考量

- [ ] 寄存器可扫描
- [ ] 时钟可控制
- [ ] 复位可控制
- [ ] 测试模式控制

### 调试支持

- [ ] 关键状态机状态可观测
- [ ] 关键计数器可观测
- [ ] 错误标志输出

### 设计文档

## 仿真验证

### 代码质量

- [ ] 无语法错误

### 功能验证
