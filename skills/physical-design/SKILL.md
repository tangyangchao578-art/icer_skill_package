# 物理设计全流程技能

当用户需要进行数字芯片物理设计时，启用此技能。提供从 floorplan 到 GDSII 全流程指导。

## 使用时机

- 全芯片物理设计
- 模块级物理设计
- 时序收敛问题调试
- 物理设计流程优化

## 物理设计流程

### 1. 输入数据准备

- 综合后门表
- 时序约束（SDC）
- 工艺库（.lef, .lib, .tech）
- IO 布局信息
- 电源地规划

### 2. Floorplan

- 设置芯片尺寸
- IO 摆放
- 电源地网络（PDN）创建
- 模块摆放
- 电源pad连接

检查点：
- [ ] 面积满足要求
- [ ] PDN 满足 IR 降要求
- [ ] 模块摆放符合数据流

### 3. Placement

- 全局 placement
- 详细 placement
- 优化 placement（时序、拥塞）

检查点：
- [ ] 布线拥塞在合理范围
- [ ] 初始时序满足初步要求
- [ ] 没有超长连线

### 4. CTS（时钟树综合）

- 时钟树目标设置（skew, latency）
- 时钟树综合
- 时钟树优化

检查点：
- [ ] 时钟 skew 满足要求
- [ ] 时钟 latency 满足要求
- [ ] 转换时间满足要求
- [ ] 功耗满足要求

### 5. Routing

- 全局布线
- 详细布线
- 布线优化
- 天线修复

检查点：
- [ ] 100% 布线完成
- [ ] 没有绕线失败
- [ ] 天线违反修复完成

### 6. Timing Closure

- 静态时序分析
- 建立时间修复
- 保持时间修复
- 迭代优化

检查点：
- [ ] 所有工艺角建立时间满足
- [ ] 所有工艺角保持时间满足
- [ ] 没有最大转换时间违反
- [ ] 没有最大电容负载违反

### 7. 物理验证

- DRC 检查
- LVS 检查
- ERC 检查
- 天线检查
- IR 降分析
- 电迁移分析

检查点：
- [ ] DRC 零违反
- [ ] LVS 零违反
- [ ] 所有检查通过

### 8. 出带

- 生成 GDSII
- 生成 LEF/DEF
- 提取 SPEF
- 生成文档

## 优化目标优先级

1. **满足时序**（建立+保持）
2. **满足DRC/LVS**
3. **满足功耗**
4. **最小化面积**

## 常见问题调试

### 拥塞过高

- 重新摆放模块
- 留出通道给大总线
- 分解大模块
- 降低局部模块密度

### 建立时间违反

- 流水线重分割
- 换更大驱动
- 复制高扇出驱动
- 重新摆放减少线长

### 保持时间违反

- 插入延迟单元
- 调整有用偏斜
- 不影响建立时间

### DRC 违反

- 检查密度填充
- 检查间距违反
- 检查最小宽度
- 根据错误信息定位修复

## 工具使用

- **布局布线**：IC Compiler II, Innovus, OpenROAD
- **时序分析**：PrimeTime, Tempus
- **物理验证**：Calibre, ICV, PVS

- 使用 `timing-analysis` 技能进行时序分析
- 使用 `drc-lvi-debug` 技能进行 DRC/LVS 调试
