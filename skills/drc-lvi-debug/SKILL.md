# DRC/LVS 调试技能

当用户需要调试 DRC/LVS 违反时，启用此技能。

## 使用时机

- DRC 违反调试
- LVS 违反调试
- ERC 违反调试
- 天线违反调试

## DRC 调试流程

### 步骤 1：理解违反类型

常见 DRC 违反：

| 违反类型 | 原因 |
|----------|------|
| 最小间距 | 两个图形距离太近 |
| 最小宽度 | 线宽小于最小值 |
| 区域密度 | 密度不符合要求 |
| 天线效应 | 长连线没有连接二极管 |
| 过孔间距 | 过孔距离太近 |
| 扩展 | 图形扩展不足 |

### 步骤 2：定位位置

- 使用工具 GUI 打开违反位置
- 放大查看周围
- 确定哪个模块哪个网络

### 步骤 3：分析原因

- 是自动布线产生的？还是手动布线？
- 是模块放置太近？
- 是Power strap 间距不对？
- 是 blockage 没有考虑？

### 步骤 4：修复

常见修复方法：

1. **间距违反**：移动图形，增加间距，或者缩小尺寸
2. **密度违反**：添加填充金属，满足密度要求
3. **天线违反**：跳线到上层金属，或者插入天线二极管
4. **过孔违反**：增加过孔间距，或者重新排列过孔

### 步骤 5：重新检查

修复后重新运行 DRC 检查，确保修复没有引入新的违反。

## LVS 调试流程

### 常见 LVS 违反

- **端口不匹配**：网表端口和版图端口数量不一样
- **节点短路**：两个应该分开的节点短路了
- **节点开路**：应该连接的节点没有连接
- **浮动节点**：节点没有连接任何东西
- **器件不匹配**：器件数量类型不匹配

### 调试步骤

1. **看错误报告**：确定哪个模块哪个网络
2. **在版图上定位**：打开 GUI 定位错误位置
3. **分析连接**：追踪连接，找出问题
4. **修复**：修改网表或修改版图
5. **重新运行 LVS**

### 常见原因

- 电源环没有正确切开分段
- 同一个名字的网络没有连接在一起
- 阻挡层挡住了连接
- P&R 输出网表层次不对
- 电源地引脚连接错误

## ERC 调试

常见 ERC 违反：

- 输入引脚没有连接
- 输出引脚没有连接
- 电源地短路
- 浮动输入引脚
- 没有连接的单元

修复方法：
- 未使用输入接地或接电源
- 浮动节点检查是否真的不需要连接
- 短路重新走线分开

## 天线效应违反

### 原因

- 长连线在下层金属，蚀刻时电荷积累，击穿栅氧
- 上层金属面积比太大，没有接到二极管

### 修复方法

1. **跳线**：从下层跳线到上层，减少暴露面积
2. **插入天线二极管**：添加二极管到栅极放电
3. **改变走线**：重新走线减少暴露面积

## 密度填充

### 要求

- 满足化学机械抛光（CMP）要求
- 每个区域密度在范围内
- 填充 dummy 金属

### 注意事项

- 填充不要增加过多电容
- 填充不影响信号完整性
- 填充遵守设计规则

## 调试技巧

1. **分层检查**：从头开始逐步检查，不要一次改很多
2. **分类处理**：同一类违反一起修复
3. **工具帮助**：使用工具自动修复大多数违反
4. **手动修复**：自动修复不了手动修复
5. **记录修复**：记录修复位置和原因方便复查

## 最终要求

流片前必须：
- DRC 零违反
- LVS 零违反
- ERC 零违反
- 天线零违反
- 密度满足要求

## 工具支持

- Calibre (Siemens EDA)
- ICV (Synopsys)
- PVS (Cadence)

## 代理协作

- 使用 `physical-design-engineer` 代理进行物理设计
- DRC/LVS 调试完成后重新检查时序
